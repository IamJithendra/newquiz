## Features

- Maze: Game mode with all other NewQuiz game modes in one quiz.

- Multi choice quiz
    - Logo quiz
    - Flag quiz
    - Solve the formula equation
    - Number trivia
- Wordle
    - Guess the text word
    - Guess the number
    - Guess the math formula
    - Number trivia
